# ModernColorOrgan
Code and info on the Modern Color Organ project.

Modern Color Organ for teensy 4.1 using Parallel output and no timer and 16 X 16 LED panels. 

Code in an .ino file : ColorOrganTeensy41ParallelNoTimer16x16Panel.ino.

SerialColorOrgan.py

SweepAudio.py
